このサイトについて

本サイトは、VBA / VB.NET / C# / SQL Server / Oracle を中心とした業務システム開発に携わるフリーランスエンジニアを対象に、主要なAIツールの実践的な活用方法を体系化したリファレンスです。表面的な操作説明ではなく、仕組みの理解・設計の根拠・業務への具体的な適用を重視しています。

各ツールのページは「概要」「初級」「中級」「上級」の4段階で構成されています。毎週月曜日 08:00 JST に最新情報を確認し、変更があった場合のみ更新します。

初級 基本操作・典型的なユースケース 中級 ツール固有機能の深掘り・他ツールとの使い分け 上級 API連携・自動化・エージェント構築・業務システム統合
🤖

ChatGPT

OpenAIが提供する汎用チャットAI。GPT-4oおよびo3モデルによる高度な推論、Advanced Data Analysis（Python実行）、カスタムGPTsによる業務特化アシスタントの構築まで幅広く対応。

概要 初級 中級 上級
⚡

Codex

OpenAIが提供するコード特化型AIエージェント。自然言語の指示からコードを生成・実行するだけでなく、既存リポジトリを解析してバグ修正・テスト追加・リファクタリングを自律的に行う。

概要 初級 中級 上級
🐙

GitHub Copilot

IDE（VS Code / Visual Studio）に統合されたAIペアプログラマー。コンテキストを読んだインライン補完、スラッシュコマンド、Copilot Workspaceによるタスク駆動開発を提供。

概要 初級 中級 上級
🔮

Claude

Anthropicが開発した、長文処理と論理的な文章生成に優れたAI。XMLタグによる構造化プロンプト、Artifacts機能、Projects機能による社内ナレッジ共有が特徴。

概要 初級 中級 上級
🦾

Manus

自律型汎用AIエージェント。サンドボックス環境でシェル・ブラウザ・コードを自律的に組み合わせ、オープンエンドな目標を達成する。GitHub連携・スケジュール実行・Webアプリ構築まで対応。

概要 初級 中級 上級
💎

Gemini

Googleが開発したマルチモーダルネイティブAI。最大200万トークンのコンテキストウィンドウ、Google Workspaceとのシームレスな統合、Vertex AIによるエンタープライズ対応が強み。

概要 初級 中級 上級
🦀

OpenClaw

オープンソースのローカルAIエージェントフレームワーク。WhatsApp・Telegram経由でAIを操作し、Webスクレイピング・DBアクセス・スケジュール実行を自動化。社内閉域網での運用が可能。

概要 初級 中級 上級

ツール選択マトリクス — ユースケース別の最適解

ユースケース 最適ツール 理由
IDE上でのリアルタイムコード補完 GitHub Copilot コンテキストスイッチなし。プロジェクト固有の変数名・命名規約を学習して提案する。
数百行のレガシーSQL / PL/SQLの解読 Claude 長文コンテキストの処理精度が高く、業務的な意図の推論と移植コードの生成を同時に行える。
複雑なアルゴリズム・数学的推論 ChatGPT (o3) o3モデルの拡張思考（Extended Thinking）は数学・論理推論において現状最高水準。
既存リポジトリへのバグ修正・テスト追加 Codex リポジトリ全体を解析した上で、複数ファイルにまたがる変更を自律的に実行できる。
複数ステップを跨ぐ「作業」の丸投げ Manus ブラウザ・シェル・コードを自律的に組み合わせ、最終成果物だけを受け取れる。
動画・画像・PDF等の多様なデータ解析 Gemini マルチモーダルネイティブ設計。200万トークンのコンテキストで大量データを一括処理。
社内閉域網での定期自動化・DB連携 OpenClaw ローカルデプロイが可能なため、社外にデータを出さずに常駐型エージェントを構築できる。
