このサイトについて

VBA / VB.NET / C# / SQL Server / Oracle を中心とした業務システム開発・病院物流システム開発に携わるフリーランスSEを主な読者として想定した個人リファレンスサイトです。

各AIツールの表面的な紹介ではなく、実際の業務システム開発・API連携・自動化に直結する技術的な深さを重視して記述しています。

各ツールページは「初級 / 中級 / 上級」の3レベルで構成されており、目的に応じて参照できます。毎週月曜日 08:00 JST に最新情報を確認し、変更があった場合のみ更新します。

Codex

Codex (OpenAI Codex CLI)

コーディング エージェント

OpenAI が提供するターミナル上で動作するコーディングエージェント。ローカルのコードベースを読み取り・変更・実行する。サンドボックス環境での自律的なコード操作が特徴。

GitHub Copilot

GitHub Copilot

コーディング エージェント

GitHub / Microsoft が提供するAIペアプログラマー。VS Code・Visual Studio・JetBrains IDEに統合され、コード補完・チャット・エージェントモードでの自律的なタスク実行が可能。

Claude

Claude

LLM コーディング

Anthropic が提供する対話型AI。Claude 4（Opus 4・Sonnet 4）は長文コンテキスト・コーディング・推論に強みを持つ。Constitutional AI による安全設計が特徴。

概要 初級 中級 上級

Manus

Manus

エージェント

Manus チームが提供する汎用AIエージェント。ブラウザ操作・コード実行・ファイル操作・Web検索を自律的に組み合わせ、複雑な多段階タスクを実行する。スキル拡張機構を持つ。

Gemini

Gemini

LLM エージェント

Google DeepMind が提供するマルチモーダルAI。Gemini 2.5 Pro/Flash は長大なコンテキストウィンドウ（100万トークン超）と強力な推論能力を持ち、Google Workspace との統合も可能。

OpenClaw
